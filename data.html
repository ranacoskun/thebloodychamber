Data Creation

After completing the modelling activity, the next step was to describe the items based on the conceptual model that was designed. This involves creating a table for each item in the form of a .csv file, where each row represents a statement about the item, and each column corresponds to the subject, predicate, and object of that statement. The subjects and objects are written in natural language, while the predicates are expressed using the ontologies that were chosen in the conceptual modelling phase.
